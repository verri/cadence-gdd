# Cadence of the Fallen — Game Design Document

A turn-based JRPG built around **timing, rhythm, and reaction**. Inspired by *The Legend of Dragoon*'s Additions, but extending the philosophy of "skill at the controller matters" into both defense and magic, so every phase of combat — attacking, defending, casting — demands presence from the player.
